Gallant
========================

A very clean Twitter Bootstrap based theme for Wordpress. 

Features: 

- Mobile device screen adjustment
- Fluid layout
- Customizable sidebar and footer
- Custom menu
- Threaded comments
- Auto table style

Live demo: [here](http://victorhurdugaci.com).
